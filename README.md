# 📋 Cadastro em PHP

Projeto simples em PHP sem uso de banco de dados.

## 🚀 Funcionalidades
- Cadastro com Nome, Email e Idade
- Visualização completa dos dados
- Visualização resumida
- Retorno mantendo os dados preenchidos

## 🛠️ Tecnologias
- PHP
- HTML
- CSS

## ▶️ Como executar
1. Instale o PHP
2. Execute:
   php -S localhost:8000
3. Acesse:
   http://localhost:8000
